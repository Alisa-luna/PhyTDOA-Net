**FM-TDOA优化定位系统**

该项目提出了一种利用现有FM广播信号进行定位的方法。该方法包含两个核心阶段：

*第一阶段：多频鲁棒粗定位*

利用每个广播塔同时发射的多个频率（如88/94/101/108 MHz）的到达时间差（TDOA）构建超定方程组。

引入自适应评分机制，根据当前环境自动归一化各频段的测量误差。

通过物理先验抑制受多径或NLOS严重污染的频段。

在预划分的地理网格上快速筛选出最可能的候选区域（Top-K）。

*第二阶段：自标定精定位*

在候选区域内，尝试引入一种带有自适应鲁棒损失函数的全局优化算法（其已在除FM定位外多个领域起到作用），联合估计接收机位置和各个广播塔的未知时钟偏差。

该损失函数能够自动降低异常测量值的权重，无需事先识别NLOS链路。

*系统优势*

基础设施零成本：利用城市中已有的FM广播信号，无需部署专用发射机。

抗干扰能力强：即使在严重多径、非视距和基站数据库误差的情况下，仍能稳定工作。

自适应性：算法能根据实时信号质量自动调整各频段和各基站的置信度。

计算高效：粗定位与精定位相结合的策略，既保证了搜索速度，又确保了定位精度。

*适用场景*

城市峡谷、室内等GPS信号弱或不可用的环境。

物联网设备、智能手机等终端的定位需求。

紧急情况下的应急定位。

**真实数据测试**

https://zenodo.org/records/4739276

*由于暂时无法获取到实测数据，且FM定位方面关于TDOA的公开数据集非常少，该项目选择OpenSky公开数据集（第一部分）进行测试*

*该数据集从分布在欧洲的323个地面传感器收录了2.22亿条飞机广播的ADS‑B信号（1090 MHz），包含每个传感器的精确到达时间戳（纳秒级）、接收功率、传感器坐标等*

*尽管它并非FM（88-108MHz），但定位原理完全匹配，也可以变相验证该方法的泛化能力*
